## 0. 使用规则

| 规则 | 说明 |

| 开工前 | 先看本文件，确认“当前阶段 / 进行中 / 下一步” |
| 完成后 | 更新状态、完成时间、备注、关联提交 |
| 提交时 | 尽量做到“一个任务一个提交” |
| 避免重复 | 若某项已标记为“已完成/不做”，不要重复开工 |
